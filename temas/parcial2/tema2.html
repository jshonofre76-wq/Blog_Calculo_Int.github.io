Tema: Funciones Racionales (Fracciones Parciales)

¿Para qué sirve este tema?

El método de fracciones parciales sirve para transformar integrales de funciones racionales complejas (cocientes de polinomios) en una suma de fracciones mucho más simples y fáciles de integrar mediante logaritmos naturales o funciones algebraicas básicas.

Caso 1: Se aplica cuando el denominador se descompone en factores lineales que son completamente distintos (no se repiten).

Caso 1: Factores Lineales No Repetidos

Problema: Resolver ∫ (5x + 3) / (x³ - 2x² - 3x) dx

Haz clic en los botones numerados para desplegar el desarrollo algebraico.
[PASO 1] Factorizar el denominador:
x³ - 2x² - 3x = x(x² - 2x - 3)
= x(x - 3)(x + 1)
Tenemos 3 factores lineales diferentes.
[PASO 2] Plantear la descomposición:
(5x + 3) / [x(x-3)(x+1)] = A/x + B/(x-3) + C/(x+1)
[PASO 3] Eliminar denominadores:
Multiplicamos todo por el denominador común:
5x + 3 = A(x-3)(x+1) + Bx(x+1) + Cx(x-3)
[PASO 4] Encontrar constantes (A, B, C):
Evaluamos en los valores críticos que hacen cero los factores:
• Si x = 0 → 3 = A(-3)(1) → A = -1
• Si x = 3 → 18 = B(3)(4) → B = 18/12 = 3/2
• Si x = -1 → -2 = C(-1)(-4) → C = -2/4 = -1/2
[PASO 5] Sustituir las constantes:
Reescribimos nuestra integral original:
∫ [ -1/x + (3/2)/(x-3) - (1/2)/(x+1) ] dx
[PASO 6] Integrar término a término:
Utilizando la fórmula ∫ dx/x = ln|x| + C:

Resultado Final:
-ln|x| + (3/2)ln|x-3| - (1/2)ln|x+1| + C
