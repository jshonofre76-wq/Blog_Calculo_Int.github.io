Aplicaciones en Cinemática

¿Para qué sirve este tema?

La cinemática estudia el movimiento de los cuerpos sin considerar las fuerzas que lo producen. Mediante el cálculo integral, podemos realizar el proceso inverso a la derivación: si conocemos la aceleración de un objeto, podemos integrar para conocer su velocidad, y volver a integrar para determinar su posición exacta en cualquier instante de tiempo.

Problema de Aplicación Física

Enunciado: Una partícula se mueve con una aceleración a = 18 - 6t². Parte de una posición inicial x₀ = 100m en t = 0 con una velocidad inicial v₀ = 0. Determine el tiempo t cuando v = 0, y calcule la posición y velocidad en t = 4s.

Haz clic en los botones para resolver las ecuaciones físicas paso a paso.
[PASO 1] De Aceleración a Velocidad (Integración):
Sabemos que a = dv/dt  → dv = (18 - 6t²)dt
∫ dv = ∫ (18 - 6t²) dt
v(t) = 18t - 2t³ + C
Aplicando condición inicial v(0) = 0 obtenemos C = 0.
Ecuación: v(t) = 18t - 2t³
[PASO 2] Encontrar cuándo la velocidad vale cero:
Igualamos a cero: 18t - 2t³ = 0
Factorizamos por término común: 2t(9 - t²) = 0
• Raíz 1: 2t = 0 → t = 0 s (Instante de partida)
• Raíz 2: 9 - t² = 0 → t² = 9 → t = 3 s
La velocidad vuelve a ser cero a los 3 segundos.
[PASO 3] De Velocidad a Posición (Segunda Integración):
Sabemos que v = dx/dt  → dx = (18t - 2t³)dt
∫ dx = ∫ (18t - 2t³) dt
x(t) = 9t² - (t⁴ / 2) + C
Como x(0) = 100m (posición inicial), entonces C = 100.
Ecuación final: x(t) = 100 + 9t² - (t⁴ / 2)
[PASO 4] Evaluar la posición en t = 4 segundos:
Sustituimos t = 4 en x(t):
x(4) = 100 + 9(4)² - (4⁴ / 2)
x(4) = 100 + 9(16) - (256 / 2)
x(4) = 100 + 144 - 128
Resultado Posición: x(4) = 116 metros
[PASO 5] Evaluar la velocidad en t = 4 segundos:
Sustituimos t = 4 en v(t):
v(4) = 18(4) - 2(4)³
v(4) = 72 - 2(64)
v(4) = 72 - 128
Resultado Velocidad: v(4) = -56 m/s  [Moverse en sentido contrario]
