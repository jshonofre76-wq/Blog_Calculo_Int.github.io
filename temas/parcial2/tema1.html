Área Bajo la Curva

¿Para qué sirve este tema?

Permite determinar la magnitud del espacio bidimensional delimitado por la gráfica de una función matemática, el eje de las abscisas (Eje X) y un intervalo cerrado dado [a, b].

Ejercicio Práctico Interactivo

Problema del Profesor: Hallar el área (A) de f(x) = x³ + 1 en el intervalo (0, 3).

Haz clic en los botones de arriba para desplegar el procedimiento interactivo paso a paso.
[PASO 1] Planteamiento del Área:
A = ₀∫³ (x³ + 1) dx
[PASO 2] Integración por separado:
∫ x³ dx = x⁴ / 4
∫ 1 dx = x
F(x) = (x⁴ / 4) + x
[PASO 3] Aplicación del Teorema Fundamental:
Evaluamos F(3) - F(0):
F(3) = (3⁴ / 4) + 3 = (81 / 4) + 3 = 20.25 + 3 = 23.25
F(0) = (0⁴ / 4) + 0 = 0

Resultado Final:
A = 23.25 unidades cuadradas (u²)
