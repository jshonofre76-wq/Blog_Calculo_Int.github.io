Caso: √(a² + x²)

Sustitución:
x = a tan(θ)

Diferencial:
dx = a sec²(θ) dθ

Identidad Raíz:
√(a² + x²) = a sec(θ)

Caso: √(x² - a²)

Sustitución:
x = a sec(θ)

Diferencial:
dx = a sec(θ)tan(θ) dθ

Identidad Raíz:
√(x² - a²) = a tan(θ)

Caso: √(a² - x²)

Sustitución:
x = a sin(θ)

Diferencial:
dx = a cos(θ) dθ

Identidad Raíz:
√(a² - x²) = a cos(θ)

📖 APÉNDICE: Construcción del Triángulo de Pitágoras

• Suma √(x² + a²): La raíz representa la hipotenusa. El opuesto es la variable (x) y el adyacente es la constante (a).
• Resta √(x² - a²): El término variable positivo (x) es la hipotenusa. La constante (a) es el adyacente. La raíz es el opuesto.
• Resta √(a² - x²): El término constante positivo (a) es la hipotenusa. La variable (x) es el opuesto. La raíz es el adyacente.

💡 Truco del Triángulo para Despejes Rápidos

Después de elegir la sustitución se construye el triángulo rectángulo. Este permite obtener el valor de la raíz y otras razones trigonométricas directamente sin necesidad de volver a sustituir algebraicamente "x" dentro de la raíz original. Se recomienda usar relaciones de seno o coseno porque tienen la hipotenusa en el denominador, facilitando los despejes necesarios.

🧮 FORMULARIO AVANZADO DE IDENTIDADES Y CÁLCULO

[Razones Trigonométricas Fundamentales]

• sen(θ) = opuesto / hipotenusa
• cos(θ) = adyacente / hipotenusa
• tan(θ) = opuesto / adyacente
• sec(θ) = hipotenusa / adyacente
• csc(θ) = hipotenusa / opuesto
• cot(θ) = adyacente / opuesto

[Identidades Trigonométricas Clave]

• sen²(θ) + cos²(θ) = 1     • sec²(θ) - tan²(θ) = 1
• 1 - sen²(θ) = cos²(θ)    • sec²(θ) = 1 + tan²(θ)
• 1 - cos²(θ) = sen²(θ)    • tan²(θ) = sec²(θ) - 1

[Derivadas para Obtener dx]

• d/dθ (sen θ) = cos θ
• d/dθ (cos θ) = -sen θ
• d/dθ (tan θ) = sec² θ
• d/dθ (sec θ) = sec θ tan θ

[Integrales Directas e Inversas Formas]

• ∫ sec²(θ)dθ = tan(θ) + C    • ∫ [1/√(1-x²)]dx = arcsin(x)+C
• ∫ csc²(θ)dθ = -cot(θ) + C   • ∫ [1/(1+x²)]dx = arctan(x)+C
• ∫ sec(θ)tan(θ)dθ = sec(θ)+C • ∫ [1/(x√(x²-1))]dx = arcsec(x)+C

Sustitución Trigonométrica

¿Para qué sirve este tema y cómo usa a Pitágoras?

Este método se aplica para resolver integrales que contienen radicales de la forma √(a² - x²), √(a² + x²) o √(x² - a²).

Utiliza el Teorema de Pitágoras de forma inversa: asociamos la raíz y la variable x a los lados de un triángulo rectángulo (cateto opuesto, cateto adyacente e hipotenusa). Esto nos permite cambiar una difícil raíz algebraica por funciones trigonométricas directas muy fáciles de integrar.
